Settings

  In project files not exist settings.py. Add it and set minimal requirments

    Set one or more language

    Set GOOGLE_RECAPTCHA_SECRET_KEY django-recaptcha

    Add mix_by_category named dictionary variable

Setup

  Add this applications in INSTALLED_APPS and migrate in order.

    1. user

    2. blacklist

    3. articlecategory

    4. initialarticle

    5. article

    imagekit, simple_email_confirmation, django_summernote

    migrate wihout app name

  Afther complete migration enter to admin panel in online-newspaper/admin and create groups stuff, moderator and admin

  Add JQuery, Bootstrap, Fontawesome and popper.js in static folder for using default templaes

static folder map

Languages

  Article multilanguage system work with settings LANGUAGES. Possible create only in existed language of settings.LANGUAGES

Template Direct

  For custom templates need to create templates schema same of existed in one direct higher and edit TEMPLATES DIRS in settings

Files and dirs

  Default avatar - in media/images/avatars/ add avatar.jpg file

  Logo - in media/web/ add logo.png file

mix_by_category

  It will mix articles by category

  Usage - set as key name of category and as value number of articles
